Cores

Existem varias formas de definirmos cores utilizando o CSS

  • Cores pré-definidas
  • Palavra chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL e HSLA

hsl(hue, saturation, lightness)
  • Hue (matiz): é o grau na roda de Cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness (luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
