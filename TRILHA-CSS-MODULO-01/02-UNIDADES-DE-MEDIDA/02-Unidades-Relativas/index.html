Unidades de medida relativas

As unidades de medida relativas são calculadas com base em unidades já conhecidas de um outro elemento, o que faz com que o resultado possa variar de acordo com o valor da propriedade do elemento no qual estamos nos baseando

  • %
  • em
  • rem
  • vw
  • vh

1- Porcentagem (%)

A porcentagem é uma unidade de medida que irá utilizar como referência o elemento pai para se basear, podendo alterar os resultados de acordo com o lugar em que está inserido.


Pixel vs porcentagem


width: 300px;
width: 50%;


Elemento pai possui 500px de largura e um font-size de 40px.


width: 300px
width: 50%;



1- Unidade de Medida em

A unidade de medida em, quando utilizada para definir o tamanho de fonte de um elemento, vai utilizar como referência o valor da propriedade font-size do elemento pai no qual está inserido.


Caso seja utilizado a unidade em para outras propriedades que não sejam o tamanho da fonte do elemento, o resultado será em relação ao tamanho da fonte do prórpio elemento em que está sendo definida a propriedade.


font-size: 1em = 16px;

font-size: 2em = 32px;

font-size: 3em = 48px;



font-size: 20px;
font-size: 2em = 40px;
font-size: 2em = 80px;


font-size: 30px;
width: 10em = 300px



3- Unidade de Medida rem

O rem(root em) tem um funcionamento semelhante à unidade de medida em, porém a diferença é que, enquanto o em utiliza o elemento pai como referência, o rem está relacionado ao tamanho da fonte do elemento raiz, chamado root, que no caso, é a tag html.

font-size: 1rem = 16px;

font-size: 2rem = 32px;

font-size: 3rem = 48px;



font-size: 20px;
font-size: 2rem = 32px;
font-size: 3rem = 48px;


font-size: 20px;
font-size: 2rem = 32px;
font-size: 2rem = 32px;


font-size: 30px;
width: 10rem = 160px

  • Melhor experiência do usuário

  • Contexto dos elementos



5- Unidades relativas: Viewport Units

Viewport é a área visível de uma página web, podendo variar de dispositivo ára dispositivo. As unidades de medida viewport, vão ser calculadas em relação ao tamanho da área em que o usuário está vendo sua página.


1vw = 1% do tamanho da largura da área visível da página

Exemplo: viewport width: 375px | 10vw = 37,5px


1vh = 1% do tamanho da altura da área visível da página

Exemplo: viewport width: 812px | 10vw = 81,2px



width: 50%;
width: 50vw;
width: 50vh;


width: 50%;
width: 50vw;
width: 50vh;


width: 10vmin;
width: 10vmax;



6- Unidades relativas: ex e ch

A unidade de medida ex irá usar o tamanho da altura do caractere X (em minúsculo) da fonte do elemento.

O valor especificado altera quando a fonte for completamente alterada.


A unidade de medida ch possui como referência o caractere 0 da fonte que está sendo utilizada no elemento

Caso a fonte do elemento seja uma fonte mono-espaçada, ao utilizar a unidade de medida ch, estamos definindo quantos caracteres devem caber no elemento.
