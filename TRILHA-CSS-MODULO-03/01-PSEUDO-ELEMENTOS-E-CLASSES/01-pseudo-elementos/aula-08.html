Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declara-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::before

Permite criar um elemento dentro do seletor que está sendo selecionado, tornando assim este elemento o seu primeiro filho.

Pseudo-elemento - ::after

Permite criar um elemento dentro do seletor que está sendo selecionado, tornando assim este elemento o seu ultimo filho.
