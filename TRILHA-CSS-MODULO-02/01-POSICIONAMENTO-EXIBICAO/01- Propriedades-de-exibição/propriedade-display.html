A Propriedade CSSdisplay

A propriedade display vai determinar como as caixas dos elementos devem se comportar no momento de serem exibidas na tela

  • block
  • inline
  • inline-block
  • none
  • flex
  • grid

Todo elemento possui um display?

Todo elemento HTML já vem com um valor de display como padrão.

Cada elemento HTML já possui, por padrão, um valor da propriedade display já definidas. A tag

, por exemplo, possui um display em bloco, enquanto a tag possui o valor inline como padrão.
